# UNMANNED UNDERWATER VEHICLE #
Heavily underdevelopment unmanned underwater vehicle (semi-submersible) platform

*Tech Stack :*
- Arduino Uno
- ELRS Receiver - ER5A
- Limit Switch - Omron V158
- Water Level Sensor
- Syringe 350mm
- NEMA Motor
